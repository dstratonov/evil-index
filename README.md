# Evil Index

**Corporate Workplace Intelligence Platform**

Rate companies by their "evil level" - an aggregated score based on employee sentiment, news reports, and public feedback from open sources.

## Concept

A "Glassdoor meets investigative journalism" platform with a dark dossier aesthetic. Each company gets an Evil Index score (0-100) with a threat verdict: **TOXIC**, **HARMFUL**, **CAUTION**, or **MOSTLY OK**.

### Features

- Animated radial Evil Index gauge per company
- Threat verdict stamps with glow effects
- Breakdown bars: Work-Life Balance, Burnout Rate, Toxic Management, Pay Fairness, Job Security, D&I
- Intelligence summaries from aggregated mock data
- Source attribution: Reddit, Glassdoor, News, LinkedIn
- Search and filter by company, industry, or verdict
- Surveillance-themed UI: scan lines, grid pulse, monospace typography

## Getting Started

```bash
npm install
npm run dev
```

Opens at http://localhost:3000

## Tech Stack

- React 18
- Vite 5
- Pure CSS-in-JS (no external UI library)
